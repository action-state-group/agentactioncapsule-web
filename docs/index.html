Status. The Agent Action Capsule profile is an individual IETF Internet-Draft (draft-mih-scitt-agent-action-capsule) — submitted for discussion, not adopted by a working group, and not a standard. It builds on the IETF SCITT and COSE drafts, which are themselves still in progress and not yet published as RFCs. RFC 9162 (Certificate Transparency 2.0) is a published RFC.
